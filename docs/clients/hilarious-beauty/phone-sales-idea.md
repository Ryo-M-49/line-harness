# 電話営業 → 無料構築 → 運用回収モデル

記録日: 2026-03-26
ステータス: アイデア段階（ひらリアス案件完了後に着手）

---

## コンセプト

飲食店・クリニックに電話で直接営業し、LINE構築を無料で提供。構築の成果を見てもらったうえで月額運用に転換する。成功パターンが確立できたら外注化・システム化して横展開する。

## フロー

1. 電話営業でアポ取得
2. 構築を無料で実施（LINE Harness + テンプレートで原価ほぼゼロ）
3. 効果を実感してもらう
4. 月額運用プランに転換

## 外注化・システム化の構想

- 電話営業 → テレアポ外注（1件100〜300円が相場）
- トークスクリプトはプロに依頼して最初から質の高いものを作る
- 構築 → 業種別テンプレートで自動化（30分〜1時間で完了を目指す）
- 運用 → Claude Codeで配信コンテンツ自動生成、自分はレビューのみ

## 注意点

- 飲食店・クリニックは営業電話慣れしている。アポ率は1〜3%が一般的
- 「無料」は警戒されることもある。トークスクリプトの質が重要
- 無料構築→運用の転換率が生命線。構築後に成果を見せる仕掛けが必要
- まずひらリアス案件で構築→運用の一連を経験してから着手
