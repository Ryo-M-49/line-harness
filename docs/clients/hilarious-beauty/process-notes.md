# 契約・手続きメモ

更新日: 2026-03-26

---

## 今回の進め方

1. DMで作業内容と金額を送る
2. OKもらう（これが実質の契約）
3. LINE Official Account Managerで管理権限をもらう
4. 作業・納品
5. 請求書を送る → 振込

## 書類

| 書類 | 対応 |
|------|------|
| 見積書 | DMでの合意で代替。正式に求められたら作成 |
| 契約書 | 今回は不要。月額運用に移行する段階で業務委託契約書を用意 |
| NDA | 今回は不要。1万円の案件でNDAは相手が引く可能性あり |
| 請求書 | 納品後に送付。freee or Misocaで作成 |

## 請求書に記載する内容

- 作業内容: LINE Harnessセットアップ + リッチメニュー切替構築
- 金額: 確定後記載
- 振込先: 自分の口座情報
- 支払期限: 納品日から2週間程度

## 注意事項

- LINE公式アカウントの管理権限（管理者 or 運用担当者）を確認する
- 実現できなかった場合は無料の旨を伝えてある
- 納品後2週間は修正・質問対応する約束
