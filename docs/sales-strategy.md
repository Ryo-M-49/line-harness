ステータス: draft

## 1. 業界ターゲット分析

### 飲食店（メインターゲット）

飲食店を最優先にした理由はシンプルで、数が圧倒的に多い。全国で約60万店、コンビニの10倍以上ある。そのうちLINE公式アカウントを持っている店は増えてきたが、ほとんどが「作っただけ」の状態で放置されている。友だち登録のQRコードをレジ横に置いてるだけ、あいさつメッセージすら初期設定のまま、という店が大半。

月額1.5〜3万円を払えるかどうかは売上規模による。ここをサブカテゴリ別に見ていく。

#### サブカテゴリ別の分析

| ------ | -------- | ---------------- | ------------------------ | -------------------- |
| カフェ    | 80〜200万  | やや重い。1.5万スタートが妥当 | 新メニュー告知、雨の日クーポン、テイクアウト訴求 | 店主がSNS好きなケースが多く話が早い  |

| ラーメン店  | 150〜350万 | 2万前後が妥当          | 限定メニュー告知、スタンプカード、待ち時間通知  | オーナーが忙しく短時間勝負        |

#### 飲食店のLINE公式アカウント現状

ざっくり3パターンに分かれる。

**パターンA: アカウントすら持っていない（全体の3〜4割）**
そもそもLINEで何ができるか知らない。「うちみたいな小さい店には関係ない」と思っている。営業では「初期費用ゼロで始められる」「ダメなら辞められる」のリスクゼロ訴求が効く。

**パターンB: 作ったけど放置（全体の4〜5割）**
ここが一番おいしい。アカウントはある、友だちも数十〜数百人いる。でもあいさつメッセージは初期設定のまま、配信は月0〜1回、リッチメニューは未設定。「今あるアカウントをそのまま活かせます。初期費用もかかりません」が刺さる。

**パターンC: そこそこ使えている（全体の1割以下）**
月に数回配信して、クーポンも出している。ただしセグメント配信やステップ配信はやっていない。ここには「今やっていることの効果を3倍にできます」という改善提案で入る。

#### 月額1.5〜3万円は高いか

飲食店の販促費は売上の3〜5%が目安。月商200万の店なら月6〜10万が販促予算になる。そのうち1.5〜3万をLINE運用に充てるのは、グルメサイトの掲載料（食べログ月1〜5万、ぐるなび月1〜10万）と比べても高くない。

ただし月商100万以下の店は厳しい。1.5万でも売上の1.5%以上になり、「効果が見えないと続けられない」と言われやすい。この層には1.5万プランで入って、早期に成果を見せることが必須になる。

### 比較対象（サブターゲット）


| 業種       | 市場規模  | 月額予算感  | LINE相性             | 飲食店との比較                                    |
| -------- | ----- | ------ | ------------------ | ------------------------------------------ |
| 美容室・サロン  | 25万店超 | 3〜5万OK | 予約リマインド、再来店促進に強い   | 販促費の余裕はある。ただしホットペッパー依存が強く、LINE移行の説得に時間がかかる |
| 整体・整骨院   | 5万店超  | 2〜3万OK | 定期通院リマインド、症状別配信    | 美容室と似た構造。リピート率がKPIなのでLINEの効果を数字で見せやすい      |
| 個人ジム     | 増加中   | 2〜3万OK | 入会後フォロー、トレーニング動画配信 | 会員数が少ない分、1人あたりLTV向上の訴求が刺さる                 |
| 歯科・クリニック | 7万院超  | 3〜5万OK | 検診リマインド、予約管理       | 予算はあるが意思決定が遅い。院長への直接アプローチが必要               |


飲食店をメインにする理由をまとめると、店舗数が多い、営業効率がいい（密集エリアを歩けば何軒も回れる）、初期無料への反応がいい（販促費にシビアだから逆に「無料」に弱い）、の3点になる。

## 2. 営業方法の具体策

### 2.1 Googleマップ営業

飲食店営業の主力チャネル。店舗の所在地、営業時間、口コミ数、写真の充実度がすべてわかるので、事前の下調べがGoogleマップだけで完結する。

#### 手順

**ステップ1: エリアを決める**
自宅から電車で30分以内、もしくは自転車で行ける範囲に絞る。飲食店は対面の信頼が命なので、「何かあればすぐ来てくれる距離感」が受注率に直結する。

**ステップ2: 検索キーワードで業態を絞る**
Googleマップで以下のキーワードを順番に検索する。

- 「居酒屋」（客単価が高く月額を払いやすい。最優先）
- 「焼肉」「焼鳥」（同上）
- 「カフェ」（SNS感度が高くLINEへの理解が早い）
- 「ラーメン」（常連化しやすくリピート施策が刺さる）
- 「パン屋」「ベーカリー」（焼き上がり通知という独自の武器がある）

**ステップ3: リストアップ基準**
以下の条件で営業先をフィルタリングする。


| 条件                         | 理由                            |
| -------------------------- | ----------------------------- |
| 口コミ50件以上                   | ある程度の来客数がある証拠。月商100万以上の可能性が高い |
| 星3.5以上                     | 評判が悪い店はそもそもLINEで何をやっても効かない    |
| 写真がオーナー投稿含む                | オーナーが集客に関心を持っている              |
| LINE公式アカウントの記載がない or リンク切れ | パターンAかBの可能性が高い                |
| 個人経営っぽい                    | チェーン店は本部決裁なので個人営業では取れない       |


**ステップ4: 1日のリスト数**
1エリアにつき20〜30件をリストアップ。このうち実際に訪問できるのは10〜15件。成約率3〜5%で計算すると、週5日営業で月に200〜300件アプローチ、6〜15件の成約が見込める。

### 2.2 飛び込み営業のトーク骨格

飲食店の飛び込みは「忙しい時間帯を避ける」が鉄則。狙うのはアイドルタイムの14:00〜17:00。

#### 30秒で切り出すトーク

> すみません、お忙しいところ失礼します。近くでLINEの運用をお手伝いしている者です。
>
> 今、飲食店さん向けにLINEの仕組みを無料で作るサービスをやっていまして。
>
> たとえば日替わりメニューの配信とか、雨の日クーポンの自動配信とか、一回作ったらあとは勝手に動く仕組みなんですけど、ちょっとだけお話聞いていただけますか？

ポイントは3つ。「無料」を早めに出す。具体的な配信例を2つ挙げる（抽象的な説明は飲食店オーナーに響かない）。長くなりそうなら「チラシだけ置かせてください」で切り上げる。

#### 興味を持ってもらえた場合の2分トーク

> ありがとうございます。簡単に説明しますね。
>
> LINEの公式アカウントって、友だち登録してもらったお客さんに直接メッセージを送れるんですけど、ほとんどの店舗さんが「作ったけど使えてない」状態なんです。
>
> うちのサービスでは、LINEの初期設定からリッチメニュー、自動返信、クーポン配信の仕組みまで、全部無料で作ります。ツールの月額費用もゼロです。
>
> 使ってみて「これいいな」と思ったら、月額の運用プランに移ってもらう形です。合わなければそのまま無料で使い続けてもらって構いません。

ここで大事なのは「月額いくら？」と聞かれたときの返し方。

> 月額は1.5万〜3万円で、プランによって配信回数とかサポート内容が変わります。ただ、まず無料で作ってみて、効果を実感してから決めてもらえればいいので、今日はLINEで何ができるかだけお伝えできればと思って来ました。

金額を聞かれたら正直に答えるが、すぐに「無料で試せる」に話を戻す。飛び込みで月額契約を取ろうとしない。

### 2.3 Instagram/Googleマップの口コミDMアプローチ

飛び込みが苦手な場合、もしくは飛び込みと並行して使うオンライン営業。

#### Instagramアプローチ

飲食店のInstagramアカウントにDMを送る。投稿を必ず2〜3個見てから、その内容に触れた上でメッセージを送る。テンプレ感が出た瞬間に無視される。

> はじめまして。〇〇エリアでLINEの運用サポートをしている〇〇です。
>
> 最近の投稿にあった△△（具体的なメニュー名や写真の内容）、すごくおいしそうで気になりました。
>
> 実は飲食店さん向けに、LINEの自動配信の仕組みを無料で構築するサービスをやっていまして。たとえば新メニューの告知やクーポンの自動配信が、一度設定すれば手間なく回る仕組みです。
>
> もしLINEの活用に興味があれば、5分くらいでご説明できるのでお気軽にご連絡ください。

送信数の目安は1日20〜30通。返信率は5〜10%程度。

#### Googleマップ口コミからのアプローチ

Googleマップの口コミにオーナーが返信している店は、集客意識が高い。こういう店のInstagramやWebサイトを探して連絡する。口コミ返信を丁寧にやっている店ほど、LINEの提案にも前向きな傾向がある。

### 2.4 無料構築から有料転換の仕組み

これが営業の肝。「無料で作ります」は簡単だが、そこから月額に移行してもらえないと事業にならない。

#### 無料構築で渡すもの

- リッチメニュー（2パターン）
- あいさつメッセージ
- 自動応答（よくある質問3〜5パターン）
- 友だち追加時のステップ配信（3通）
- QRコード付き卓上POP（印刷用PDF）

これだけで「おっ、ちゃんとしてるな」と感じてもらう。ここでの品質が月額転換率を左右する。

#### 有料転換のタイミング

無料構築から2〜4週間後に「効果報告ミーティング」を設定する。ここで以下のデータを見せる。

- 友だち追加数の推移
- あいさつメッセージの開封率
- ステップ配信の反応率
- 自動応答の利用回数

「2週間で友だちが30人増えました。このペースだと3ヶ月で200人です。ここにクーポンを配信したら、月に5〜10人の追加来店が見込めます」と、具体的な数字で話す。

#### 「気に入った」と判断させるトリガー

オーナーが「これは続けたい」と思うのは、以下のどれかが起きたとき。

1. **常連客から「LINEのクーポン見ました」と言われた**。これが一番強い。実際に来店につながった実感がある
2. **友だち数が100人を超えた**。「100人に一斉にメッセージ送れるのか」という気づきが生まれる
3. **自動応答で「営業時間は？」の電話が減った**。目に見える業務効率化

この3つのうちどれか1つでも起きれば、月額転換の確率は格段に上がる。逆に2〜4週間で何も起きなかった場合は、友だち追加の導線を見直すか、配信内容を変えるか、正直に撤退するかの判断になる。

### 2.5 1人で回せる営業量の目安


| 活動                   | 1日あたり   | 1週間あたり   | 月あたり            |
| -------------------- | ------- | -------- | --------------- |
| Googleマップのリサーチ+リスト作成 | 30分     | 2.5時間    | 10時間            |
| 飛び込み営業               | 10〜15件  | 50〜75件   | 200〜300件        |
| Instagram DM         | 20〜30通  | 100〜150通 | 400〜600通        |
| 無料構築の作業              | 1件2〜3時間 | 2〜3件     | 8〜12件           |
| 効果報告ミーティング           | 1件30分   | 2〜3件     | 8〜12件           |
| 月額クライアントの運用          | 1件30分   | —        | 10〜15件で月15〜22時間 |


初月は営業8割、構築2割。3ヶ月目以降は営業5割、構築+運用5割くらいの配分になる。月額クライアントが15件を超えたあたりで、新規営業のペースを落として運用に比重を移す。

### 3.3 飲食店が「高い」と感じないか

正直なところ、月商100万以下の店にとって月額3万は高い。ここで松竹梅の「松」が効く。月1.5万なら「食べログの最安プランと同じくらいか」という感覚になる。

ただし重要なのは「安さ」で売らないこと。安さで入ると「もっと安くして」の交渉が永遠に続く。売るのは「LINEで毎月5〜10人の追加来店が生まれる仕組み」であって、「安い運用代行」ではない。

月の追加来店が5人で、客単価が3,000円なら月1.5万円の売上増。10人なら3万円。月額1.5〜3万の運用費はこの追加売上でペイする、という話を具体的にできるかどうかが分かれ目になる。

### 3.4 解約防止策


| 施策                | 内容                                                                        |
| ----------------- | ------------------------------------------------------------------------- |
| 効果の見える化           | 月次レポートで「LINE経由の来店数」「クーポン利用数」を報告する。数字が見えないと「何に払ってるかわからない」になる               |
| 季節イベントの先回り提案      | お花見、GW、七夕、お盆、ハロウィン、クリスマス、年末年始。飲食店にとってこれらは稼ぎ時。2週間前に配信プランを提案すると「任せてよかった」になる |
| 友だち数の積み上げ         | 友だちが増えるほど、LINEをやめるのがもったいなくなる。月次で「今月+30人、累計280人」のように伸びを見せる                 |
| 3ヶ月更新のタイミングでアップセル | ライトで入った店に3ヶ月目で「セグメント配信も始めませんか？」とスタンダードへの移行を提案                             |
| 配信を止めない           | 最大の解約理由は「効果がわからない」。配信を止めると効果もゼロになるので、最低でも月2回は配信を続ける                       |

### 4.2 提案書テンプレート（居酒屋の例）

提案書はA4で3枚。長くしない。飲食店オーナーは忙しいので、3分で読める量にまとめる。

**1枚目: 現状と課題**

- 「御店舗のLINE公式アカウントの現状」（友だち数、配信頻度、リッチメニューの有無）
- 「同エリアの競合店のLINE活用状況」（2〜3店舗の簡単な調査結果）
- 飲食店全体のLINE活用トレンド（友だち経由の予約が増えている等）

**2枚目: 提案内容**

- LINE Harnessで実現できること（自動配信、クーポン、予約連携、セグメント配信）
- 具体的な配信スケジュール案（月4回の配信内容イメージ）
- 想定される効果（友だち数の増加ペース、クーポン利用率の目安）

**3枚目: 料金と進め方**

- 初期構築: 無料
- 月額プランの松竹梅（スタンダードを推奨と明記）
- スケジュール（構築1〜3日 → 2週間の無料トライアル → 効果確認 → 月額契約）

## 補足: 飲食店業態別の配信アイデア集

営業時に「具体的に何を送るの？」と聞かれたときに即答するための引き出し。

**居酒屋**

- 金曜の昼に「今週末のおすすめ」を配信
- 雨の日に「雨の日限定1杯無料」クーポン
- 宴会シーズン前にコース料理の案内
- 常連向けにバースデークーポン（タグで誕生月を管理）

**カフェ**

- 新作ドリンクの写真付き告知
- 平日午後の空いてる時間帯にテイクアウト割引
- 季節限定メニューのカウントダウン配信

**ラーメン店**

- 限定麺の告知（「あと30杯で終了」の緊急感）
- 空いてる時間帯のお知らせ（待ち時間を嫌う客に刺さる）
- トッピング無料クーポン（来店のきっかけ作り）

**パン屋**

- 焼き上がり通知（「クロワッサン、10:30に焼き上がります」）
- 閉店前の値引き情報
- 予約パンの受付開始通知

**焼肉・焼鳥**

- 記念日クーポン（誕生日、結婚記念日）
- 平日限定のコース割引
- 新しい仕入れ先の肉の紹介（ストーリー性のある配信）
