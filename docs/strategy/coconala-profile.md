========================================
プロフィール本文
========================================

プロフィールをご覧いただきありがとうございます。ソフトウェアエンジニアとして5年以上の実務経験があり、バックエンドのAPI開発・データ処理・システム自動化を専門としています。

現在はEC運営サポートとLINE公式アカウントの構築・運用に注力しています。

EC領域では、ショップ運営の実務サポートからツール開発まで対応しています。LINE領域では、美容室やサロン、整体院などの店舗ビジネス向けに、ステップ配信・セグメント配信・リッチメニュー・BAN対策までを含むLINE自動化の構築を行っています。独自のオープンソースシステムを使うため、Lステップ等の有料ツールが不要で、ツール月額0円で運用できる仕組みを提供しています。


【 対応できること 】

▍LINE公式アカウント構築・運用
ステップ配信の設計と設定（友だち追加後の自動フォロー）
セグメント配信・タグ設計（属性ごとに最適な配信）
リッチメニューの作成と切替設定
自動応答・キーワード応答の設定
BAN対策（5分おきの自動監視+異常時アラート）
LINE活用の診断レポート作成

▍EC運営サポート
商品リサーチ・データ入力・商品登録
競合調査・トレンド分析
出品作業の代行・運営事務全般

▍ツール開発・業務自動化
商品データの自動収集・整形ツール
出品作業の効率化スクリプト
API連携による業務フロー自動化


【 得意な技術 】
Go / TypeScript / Python / PHP / GAS / Cloudflare Workers


【 大切にしていること 】
正確で丁寧な作業を心がけています
チャットは24時間以内に返信いたします
手作業で終わらせず「次も楽になる仕組み」を考えます

EC運営でも、LINE活用でも、「この作業、毎回手間がかかっている…」というお悩みがあれば、お気軽にご相談ください。
