LINE公式アカウントを作った。あいさつメッセージだけ設定した。それっきり止まっている。

月に1回、思い出したように配信してみるものの、何を送ればいいのかわからない。ホットペッパーには毎月手数料を払い続けているのに、LINEで友だちになったお客様には何もできていない。

構築代行を調べてみたら30万円以上。ツール代だけでも月2万円。うちみたいな個人サロンには現実的じゃない。自分でやろうにも設定画面は複雑で、スタッフもITに強いわけではない。結局、アカウントだけが残っている。

こういうお店のためのサービスです。


同じ悩みを何十件も聞いてきました。やりたい気持ちはあるのに、費用と手間の壁で止まっている。小さなサロンほどその壁は高くなる。

当サービスでは独自のオープンソースシステムを使っています。一般的なLINE拡張ツールは月額1万から3万円かかりますが、このシステムならツール利用料は0円です。配信コンテンツの下書き、データ分析、レポート作成にはAIを組み込んでいて、人件費を圧縮しているぶん、運用の質を落とさず価格を抑えています。


やることは3つだけです。

1つ目。LINEかZoomで30分ほどお話を聞かせてください。お店の特徴、メニュー、客層、今困っていること。それだけで十分です。

2つ目。ヒアリングをもとに、タグ設計、シナリオ設計、リッチメニュー作成、自動応答設定をこちらで進めます。3日から5日で完成します。テスト用アカウントで動作を確認いただき、気になる点があればこの段階で修正します。

3つ目。本番アカウントに適用して運用開始。月額プランの場合は、ここから毎月の配信と改善が続きます。


LINE周りの設定はすべてこちらで行います。お店側にお願いするのは、LINE公式アカウントを開設することと、QRコードを店頭に置くこと。この2つだけです。


もしこのまま放置を続けたらどうなるか。ホットペッパーへの依存は変わらず、手数料は毎月出ていく。来店してくれたお客様との接点はレジで終わり、次の来店を待つしかない。せっかく友だち追加してくれた人たちは、何のメッセージも届かないまま、お店の存在を忘れていく。


導入するとお店の日常はこう変わります。

友だち追加した瞬間に初回限定クーポンつきのあいさつが届く。翌日にサロンの紹介。3日後に髪の悩み相談の呼びかけ。1週間後にクーポンのリマインド。2週間後にセットメニューの提案。一度設定すれば、この5通が自動で届き続けます。

来店後は自動でお礼メッセージを送り、次の予約を促す。しばらく来ていないお客様にはクーポンを配信して呼び戻す。

ホットペッパー経由の予約をLINE予約に移行すれば手数料が減ります。リピーターとの関係はLINEで完結させて、広告費に頼らない集客の土台ができる。

お店は施術に集中して、LINEは裏で静かに動き続ける。そういう仕組みです。


料金は3つのプランがあります。

初期構築プラン 59,800円。友だち追加時の自動あいさつ、ステップ配信5通の設計と設定、リッチメニューの作成、顧客タグの設計、自動応答の設定、納品後30日のチャットサポートが含まれます。代行業者に依頼すると30万から250万円かかる作業です。

月額運用プラン 29,800円。月4回から8回の配信代行、セグメント配信の設計と実行、リッチメニューの季節対応、月次レポート、シナリオの効果測定と改善、BAN監視を含みます。通常は月10万から50万円の運用を、AIによる自動化で実現しています。

スタートパック 79,800円。初期構築と月額運用3ヶ月分のセットです。3ヶ月で土台を作り、成果が出てから続けるかどうか判断できます。


既存ツールとの費用比較も載せておきます。

構築費用はLステップ構築代行が30万から250万円、当サービスは59,800円。ツール月額はLステップが21,780円から、当サービスは0円。月額運用費は運用代行が10万から50万円、当サービスは29,800円。構築期間は業界相場が2週間から4週間、当サービスは最短3日。追加変更は都度見積もりが一般的ですが、月額プランに含んでいます。BAN検知機能は既存ツールにはありませんが、当サービスでは5分おきの自動監視が標準でついています。


いくつか補足です。

月額プランはいつでも解約できます。構築した設定やデータはそのまま残るので、自分で運用に切り替えることもできます。

Lステップなど他のツールからの乗り換えにも対応しています。既存のシナリオや顧客データを確認したうえで移行します。

BAN対策はシステムが5分おきにアカウントの状態を自動チェックしています。異常を検知したらすぐに連絡し、配信停止やアカウント切替の対応をします。


まずは30分、お店の話を聞かせてください。LINEかZoomで無料相談を受け付けています。
