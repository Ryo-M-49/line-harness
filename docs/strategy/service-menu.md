# LINE運用サービスメニュー

作成日: 2026-03-26
ステータス: draft
用途: 価格提示・DM営業・ヒアリング後の提案

---

## 共通事項

- ツール月額: **0円**（OSSのLINE Harnessを使用。Lステップ等の月額2〜3万円が不要）
- LINE公式アカウント本体の料金（0円/5,000円/15,000円）は別途お客様負担
- 契約期間の縛りなし。月単位で解約可能

---

## 構築パッケージ（単発）

初期構築のみ。運用は自分で行いたい方向け。

| 内容 | 詳細 |
|------|------|
| LINE公式アカウント初期設定 | プロフィール、あいさつメッセージ |
| リッチメニュー作成・設定 | デザイン1パターン、タップアクション設定 |
| ステップ配信シナリオ | 5〜10通の自動配信シナリオ設計・設定 |
| セグメント・タグ設計 | 業種に合わせた分類設計 |
| 自動応答設定 | よくある質問への自動回答 |
| 納品後サポート | 30日間チャットで質問対応 |

**価格**: 5万〜15万円（内容により変動）
**納期**: 3〜5日

---

## 月額運用プラン

### ライトプラン（月額1万円）

配信だけお任せしたい方向け。最低限の「動いている状態」を維持する。

| 内容 | 詳細 |
|------|------|
| 配信代行 | 月4回（週1回ペース） |
| 配信コンテンツ作成 | テキスト+画像の構成提案・作成 |
| リッチメニュー管理 | 軽微な修正・差替え（月1回まで） |
| BAN監視 | 自動ヘルスチェック、異常時アラート通知 |
| チャットサポート | 月2回まで質問・相談対応 |

**含まれないもの**: セグメント配信の最適化、シナリオ改善、レポート、戦略提案

---

### スタンダードプラン（月額3万円）

配信+改善を回したい方向け。データを見ながら毎月やることを調整していく。

| 内容 | 詳細 |
|------|------|
| 配信代行 | 月8回（週2回ペース） |
| 配信コンテンツ作成 | テキスト+画像+Flexメッセージ対応 |
| セグメント配信 | タグ・属性ベースの配信先出し分け |
| リッチメニュー更新 | 季節・キャンペーンに合わせた差替え（月2回まで） |
| シナリオ改善 | ステップ配信の分岐見直し・A/Bテスト |
| 月次レポート | 友だち数推移、配信成果、改善提案を1枚にまとめて共有 |
| BAN監視 | 自動ヘルスチェック+リスク時アラート |
| チャットサポート | 無制限 |

---

### プレミアムプラン（月額5〜10万円）

LINEを売上の柱にしたい方向け。戦略から実行・分析まで全部任せられる。

| 内容 | 詳細 |
|------|------|
| スタンダードの全内容 | — |
| 月1回の戦略ミーティング | 30分〜1時間。施策方向性・KPI確認 |
| スコアリング運用 | 見込み度に応じた自動振り分け・ホットリード通知 |
| CV計測 | 流入元ごとの成果追跡（どの施策が売上に効いているか） |
| 外部連携 | 予約システム・決済（Stripe等）とのWebhook連携 |
| リマインダー配信 | 予約前日通知、セミナーカウントダウン等 |
| BAN時の移行対応 | 友だちデータの新アカウントへの自動引き継ぎ |
| リッチメニュー更新 | 回数無制限 |
| 配信回数 | 無制限（必要な分だけ） |
