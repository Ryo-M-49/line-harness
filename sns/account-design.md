作成日: 2026-03-31
ステータス: 設計中

---

## 目的

1. **営業の名刺代わり**: Google Maps営業・飛び込み時に「この人LINE詳しい」の信頼感を即座に伝える
2. **インバウンド集客**: 飲食店オーナーが自発的に問い合わせてくる導線を作る
3. **ブランド構築**: 「小規模飲食店 × LINE公式」の第一想起ポジションを取る

---

## ポジショニング

### 狙うポジション

> **「Lステップ不要。LINE公式アカウントの無料〜ライト機能だけで、リピート型の小規模店舗（飲食店・美容室・整体・クリニック等）の売上を上げる方法を発信する人」**

### ターゲット業種

| 業種 | 店舗数（全国） | LINE相性 | コンテンツでの扱い |
|------|-------------|---------|----------------|
| 飲食店（居酒屋・レストラン・カフェ・ラーメン等） | 約60万軒 | ◎ クーポン・空席・新メニュー告知 | メイン事例 |
| 美容室・サロン | 約25万軒 | ◎ 予約リマインド・指名促進 | サブ事例 |
| 整体・マッサージ・鍼灸 | 約14万軒 | ○ 定期来院促進・症状別配信 | サブ事例 |
| クリニック・歯科 | 約10万軒 | ○ 予約管理・定期健診リマインド | サブ事例 |
| ジム・フィットネス | 約5万軒 | ○ 退会防止・イベント告知 | ローテーション |
| パン屋・ケーキ屋 | 約5万軒 | ◎ 焼き上がり通知・予約 | ローテーション |

**合計: 約120万軒+** が潜在ターゲット（飲食店オンリーの2倍以上）

### なぜこのポジションが空いているか

| 既存プレイヤー | 彼らが来られない理由 |
|--------------|-------------------|
| Lステップ認定トレーナー（中村誠、堤建拓等） | Lステップ月3.3万円が前提。「無料でできる」は自社商品の否定になる |
| LINE構築代行会社 | 月20万円〜の高単価コンサルが収益源。月1.5万円の層は利益が出ない |
| ツール公式（Lステップ、L Message、Liny） | ツール販売が目的。「ツール不要」とは言えない |
| LINEヤフー公式 | チェーン店事例が中心。個人経営の泥臭い運用ノウハウは出しにくい |

### ポジショニングマップ上の位置

```
        【高単価・プロ向け】
             |
   Lステップ系     構築代行会社
             |
【ツール前提】──────+──────【ツール不要】
             |
   ツール公式      ★ここを取る ★
             |          リピート型の小規模店舗全般
        【低単価・店舗オーナー向け】
```

## アカウント設計

### アカウント名案

| 案 | 意図 |
|----|------|
| **LINE集客ラボ｜小さなお店の集客術** | 「小さなお店」で飲食・美容・整体を包含。検索性+親しみやすさ |
| **LINE集客ラボ｜街のお店専門** | 「街のお店」で地域密着の小規模店舗全般をカバー |
| **LINE集客ラボ｜個人店の味方** | 応援感・寄り添い感が出る。フォローの心理的ハードルが低い |

**推奨**: 「LINE集客ラボ｜小さなお店の集客術」（ターゲットが広い + 検索性 + 何を得られるか明確）

### プロフィール文

```
📱 小さなお店のLINE公式アカウント運用を支援
🏪 飲食店・美容室・整体・クリニック…リピーターが大事なお店向け
💰 ツール代0円・月1.5万円〜で常連客を増やす方法
🔧 構築無料 → 効果が出たら月額サポート
✅ 累計○○店舗の構築実績
👇 無料相談はLINEから
```

### プロフィールリンク

LINE公式アカウント（LINE Harnessで構築した自社アカウント）への誘導
→ 友だち追加で「小さなお店のLINE運用チェックリスト」を自動配信

### アイコン・ビジュアル

- アイコン: LINEグリーン + 店舗モチーフ（吹き出し + 店のシルエット）
- トンマナ: 緑基調 + 白 + 暖色系アクセント。清潔感あるフラットデザイン
- フォント: 太めのゴシック体。数字を大きく見せる

## プラットフォーム戦略

### メイン: Instagram + TikTok（リール同時投稿）

| 項目 | Instagram | TikTok |
|------|-----------|--------|
| 役割 | 名刺・信頼構築・保存されるコンテンツ | リーチ拡大・バイラル獲得 |
| 形式 | リール + カルーセル投稿 | ショート動画のみ |
| 投稿頻度 | 週5本（リール3 + カルーセル2） | 週3-5本（リールと同じ動画を転載） |
| ターゲット | 小規模店舗オーナー（30-50代） | 店舗オーナー + 開業準備中の層 |
| CTA | プロフィールリンク→LINE公式 | プロフィールリンク→Instagram or LINE |

### サブ: X（Twitter）

| 項目 | 内容 |
|------|------|
| 役割 | テキストベースの深い情報発信 + 開発者/同業者とのネットワーク |
| 形式 | 単発ツイート + スレッド |
| 投稿頻度 | 毎日1-2ツイート |
| 内容 | 構築事例の数字公開、運用Tips、飲食店LINE界隈のニュース |

### 運用フロー

```
① 週初めにネタ5-7本を企画
② 動画撮影（画面録画 or 実店舗撮影）
③ TikTok + Instagram リールに同時投稿
④ カルーセル版をInstagramに追加投稿
⑤ 要約をXに投稿
⑥ 週末に数値振り返り
```

## コンテンツ方針

### コンテンツの4軸

| 軸 | 割合 | 例 |
|----|------|---|
| **教育系**（How-to） | 40% | 「クーポン設定の手順」「リッチメニューの作り方」 |
| **実績・事例系** | 25% | 「導入3ヶ月で再来店率2倍の居酒屋」「美容室の指名率UP」 |
| **共感・あるある系** | 20% | 「小さなお店のオーナーの9割がやってるLINEのミス」 |
| **ブランディング系** | 15% | 「なぜ無料で構築するのか」「LINE運用の裏側」 |

### フォーマット

**リール/TikTok（30-60秒）**:
1. フック（3秒）: 問題提起 or 衝撃の数字
2. 本題（20-40秒）: 具体的なやり方 or 事例
3. CTA（5秒）: 「他にも飲食店LINE運用のコツはプロフィールから」

**カルーセル（5-10枚）**:
1. 表紙: キャッチーなタイトル
2. 問題提起: 「こんなことありませんか？」
3. 解決策: 3-5ステップ
4. 実例: スクリーンショット付き
5. まとめ + CTA

### トーン

- 専門用語は使わない（「ステップ配信」→「自動で順番にメッセージを送る機能」）
- 飲食店オーナーの言葉で話す（「CVR」ではなく「来てくれたお客さんの数」）
- 上から教えるのではなく、一緒に考える姿勢
- 実際の画面を見せながら説明（エアプ感を出さない）

## 営業との連携

### 名刺への記載

```
[表面]
LINE集客ラボ｜小さなお店の集客術
[名前]
小規模店舗のLINE公式アカウント運用支援

[裏面]
Instagram: @line_shukaku_lab（QRコード）
LINE: @xxx（QRコード）
「フォローでお店のLINE運用チェックリストをプレゼント」
```

### 営業導線

```
Google Maps営業 / 飛び込み
  → 名刺渡す「Instagramでお店のLINE運用のコツを発信してます」
  → オーナーがアカウントを見る（投稿が信頼の証拠になる）
  → 「無料で構築しますよ」の提案に説得力が出る
  → LINE友だち追加 → 自動ステップ配信で教育
  → 月額サポート契約
```

### Instagram DM営業との連携

```
① 地域の店舗アカウント（飲食・美容・整体等）をフォロー
② 投稿にコメント（営業感なし、純粋な感想）
③ 相手がフォロバ or 反応したらDM
④ 「LINE公式やってますか？無料で構築してるんですが」
⑤ 自分の投稿が実績の証明になる
```
