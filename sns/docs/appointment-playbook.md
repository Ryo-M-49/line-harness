# 初回商談プレイブック — アポ取得後の流れ

作成日: 2026-03-31
ステータス: draft

---

## 1. スライドは使わない（重要な方針）

小規模店舗のオーナー相手にノートPCを開いてスライドを見せると、一気に「営業されてる感」が強くなる。特に飲食店、美容室、整体、クリニックの現場では、カウンターや受付横の数分で話すことが多く、プロジェクターもモニターもない。そこでパワポを出すと場に合わない。

初回商談で使うのは、以下の4つで十分。

- **デモ用LINE公式アカウント**。スマホで実際に触ってもらう。これが最強の営業ツール
- **A4チラシ 1枚**。置いていくもの。スライドの代わり
- **卓上POP見本**。実物を見せると「うちの店にも置きたい」になりやすい
- **タブレット or スマホ**。デモ画面を見せる用

商談は「説明」より「体験」に寄せた方が強い。隣に座ってスマホ画面を一緒に見てもらう方が自然だし、相手も理解が早い。提案書っぽいものが必要なら、あとでPDFを送れば足りる。

> 実際に見てもらった方が早いので、スマホで触ってみてもらっていいですか

> 資料を読むというより、お客さん目線で体験してもらう感じです

## 2. 商談前日までの準備

初回商談の勝率は、その場の話し方より準備で決まる。目安として、リサーチ15〜20分、デモ準備10分。ここを雑にすると、商談中の提案がぼやける。

### 店舗リサーチ（15-20分）

- [ ] Googleマップで口コミ・写真・営業時間を確認する
- [ ] Instagramをチェックする。投稿内容、フォロワー数、最近の更新頻度を見る
- [ ] LINE公式アカウントがあれば友だち追加して現状を確認する
- [ ] あいさつメッセージ、リッチメニューの有無、最近の配信状況を見る
- [ ] 食べログ、ホットペッパー、EPARKなどの掲載有無とプラン感を確認する
- [ ] 周辺の競合店でLINEを活用している店があるかチェックする

ここで見たいのは「この店が何に困っていそうか」と「今すでに何にお金を使っているか」の2点。Instagramが止まっていれば発信が続かないタイプかもしれないし、口コミ返信を丁寧に返していれば集客意識は高い。

> Instagramかなり丁寧に更新されてますよね。ここにLINEがつながるとかなり強いです

> 競合の○○店さんはLINEでクーポン配ってましたけど、御店はまだ伸びしろあります

### デモ準備（10分）

- [ ] デモ用LINE公式アカウントの動作確認をする
- [ ] 業態に合わせてデモシナリオを切り替える
- [ ] 飲食ならクーポン中心、美容なら予約リマインド中心、整体なら定期来院中心、クリニックなら検診・再来院リマインド中心で組む
- [ ] 相手の店名を入れた仮のリッチメニューをCanvaで1枚作っておく

この「御店用に作ってみました」が一番刺さる。汎用デモを見せるだけだと他人事で終わるが、店名が入るだけで急に自分ごとになる。

> まだ仮ですけど、御店だったらこんな見せ方が合うかなと思って1枚作ってきました

### 持ち物チェックリスト

- [ ] スマホ（デモ用LINE公式アカウントにログイン済み）
- [ ] タブレット or サブスマホ（相手に渡して触ってもらう用）
- [ ] 名刺（SNSアカウントのQR付き）
- [ ] A4チラシ 2-3枚（置いていく用）
- [ ] 卓上POP見本（実物）
- [ ] 相手の店名入りリッチメニュー画像（スマホに保存）
- [ ] モバイルバッテリー

## 3. 商談当日の流れ（合計15-20分）

初回商談は長くやらない。15〜20分で十分。ここでのゴールは契約ではなく、無料構築に進む合意を取ること。

### Step 1: アイスブレイク（2-3分）

店に入ったら、まず「お客さんとして」の視点で一言入れる。いきなり営業トークに入らない。

- 店に入ったらまず感想を伝える
- 電話のお礼を伝える
- カウンターかテーブルの「隣」に座る
- 対面に座らない。対立構造に見えやすいから
- 名刺を渡し、SNSアカウントにも軽く触れる

セリフ例:

> 外観すごくいいですね。さっきメニュー見ましたけど、○○気になりました

> 先日はお電話でお時間いただいて、ありがとうございます。今日は5分でも10分でも大丈夫です

> ここでお店向けのLINE運用のコツも発信してるので、あとでよければ見てみてください

### Step 2: ヒアリング（3-5分）

ここは深掘りしすぎない。短時間で現状をつかんで、A/B/Cのどのパターンかを判定する。

- 集客の現状を聞く
- LINEの現状を聞く
- 困っていることを引き出す
- 販促費の感覚を探る
- この段階でパターンA/B/Cを判定する

聞くことの例:

> 今ってお客さんはどうやって来てますか

> リピーターさんってどれくらいの割合ですか

> LINE公式アカウントって今お持ちですか

> 友だち数ってだいたい何人くらいですか

> 配信って最近されていますか

> 集客で一番困ってることって何ですか

> 食べログとかホットペッパーって使ってますか。月額ってどれくらいですか

パターン判定の目安:

- **パターンA**: LINE公式アカウントがない
- **パターンB**: 持っているが放置
- **パターンC**: ある程度使っている

この判定が後半の提案の角度を決める。Aには「ゼロから無料で作れる」、Bには「今あるものを活かせる」、Cには「今の運用を改善できる」で入る。

### Step 3: デモ（5-7分）— ここが勝負

ここが商談の本体。説明ではなく、相手に触ってもらう。

- 「実際に見せた方が早い」と言ってスマホを出す
- デモ用LINEアカウントのQRを読んでもらう
- その場で友だち追加してもらう
- あいさつメッセージを見せる
- リッチメニューを見せる
- クーポンや予約導線を実際に操作して見せる
- 相手の店名入りリッチメニュー画像を見せる
- 興味を示した機能を深掘りする

セリフ例:

> ちょっと説明するより、実際に触ってもらった方がわかりやすいです

> これがお客さんが最初に見る画面です。最初の印象はここで決まります

> ここから予約、クーポン、メニューに飛べます。お客さんはボタン押すだけです

> 御店だとこんなイメージです。まだ仮ですが、これ無料で作れます

相手の反応が強いポイントを見逃さない。「これいいですね」が出た機能が、そのまま次の提案軸になる。飲食ならクーポン、美容なら予約導線、整体ならリマインド、クリニックなら受診導線や検診告知が反応されやすい。

### Step 4: 提案（3-5分）

デモのあとに、ヒアリング内容を踏まえて「御店だったらこう使える」を3つだけ提案する。数を出しすぎると逆にぼやける。

- ヒアリング結果を踏まえて3つ提案する
- 業態別に使い方を具体化する
- 料金の話は「まず無料で作って試す」に戻す
- 聞かれたら月額レンジを伝える
- A4チラシを渡す

業態別の具体例:

- 飲食: 雨の日クーポン、週末おすすめ配信、バースデークーポン
- 美容: 次回予約リマインド、指名紹介、空き枠告知
- 整体: 施術後フォロー、定期来院リマインド、季節の健康情報
- クリニック: 検診リマインド、休診日案内、再診フォロー

セリフ例:

> 御店だったら、まずはこの3つから始めるのが一番効くと思います

> 初期構築は全部無料です。まず作ってみて、2-3週間使ってみて、効果を見てから月額プランを考えてもらえれば大丈夫です

> 月額は1.5万〜3万円で、配信回数やサポート内容で変わります。ただ、食べログやホットペッパーの月額と比べるとだいぶ軽いと思います

> 詳しい内容はこの1枚にまとめてます。後でスタッフさんにも見せやすいと思います

### Step 5: クロージング（2-3分）

初回商談のゴールは契約ではない。無料構築の合意を取ること。この着地をぶらさない。

- ゴールは「無料構築の合意」
- 管理者権限の共有を依頼する
- 次回アポをその場で決める
- LINEでいつでも聞けると伝える

セリフ例:

> よければ来週中に一式こちらで作ってお持ちしましょうか。必要なのは、LINEの管理者権限を共有してもらうことだけです

> 設定画面から追加するだけなので、1分で終わります

> 次は作ったものをお見せしながら、使い方もその場で一緒に確認しましょう

> 何かあればLINEでいつでも聞いてください

## 4. 商談後のアクション（当日〜翌日）

商談は、終わってからの動きで印象が決まる。その日のうちに連絡しないと、熱が落ちる。

### 当日中

- [ ] LINE or メールでお礼メッセージを送る
- [ ] 商談メモを記録する
- [ ] 無料構築が合意できた場合は管理者権限の共有を依頼する

お礼メッセージのテンプレ:

アポ取れた場合:

> 本日はお忙しい中ありがとうございました。お話いただいた内容をもとに、○○を来週までに作成してお持ちします。途中で確認事項があればLINEでご連絡しますので、引き続きよろしくお願いします。

保留の場合:

> 本日はお時間いただきありがとうございました。今日お見せした内容でご不明点があれば、いつでもご連絡ください。必要であればスタッフの方向けにも改めてご説明できます。

商談メモで最低限残すこと:

- 店名
- オーナー名 or 店長名
- パターンA / B / C
- 反応が良かったポイント
- 懸念点
- 次回アクション

### 翌日〜構築完了まで（1-3日）

- [ ] LINE Harnessで構築を開始する
- [ ] リッチメニュー2パターンを作る
- [ ] あいさつメッセージを設定する
- [ ] 自動応答を3-5パターン作る
- [ ] ステップ配信を3通作る
- [ ] QR付き卓上POPをPDFで作る
- [ ] 中間報告を1回送る

中間報告のテンプレ:

> 順調に進んでいます。現在、リッチメニューとあいさつメッセージまで作成できています。明日までに残りの設定も整えて、次回お見せできる状態にします。

放置している印象を出さないことが大事。1通あるだけで安心感が変わる。

### 納品日（構築完了後）

- 2回目の訪問で納品と使い方説明をする
- オーナーと一緒にスマホでリッチメニューやクーポンを確認する
- 卓上POPの設置場所を一緒に決める
- 効果測定期間を2-3週間で設定する
- 効果報告ミーティングのアポをその場で取る

セリフ例:

> ここから2-3週間で、友だち追加数やクーポンの反応を一緒に見ていきましょう

> レジ横、テーブル、入口のどこに置くのが一番自然か、一緒に見てもいいですか

## 5. 効果報告ミーティング（2-3週間後）

ここで初めて月額プランの提案を本格化する。感覚ではなく、数字で話す。

- 友だち追加数を報告する
- あいさつメッセージの開封率を報告する
- 自動応答の利用回数を報告する
- クーポン利用数を報告する
- 具体的な数字から将来の伸びを示す
- まずはスタンダード（2.5万円）を推奨として出す
- 断られたらライト（1.5万円）を提案する
- それでもNOなら無料のまま使ってもらい、後日フォローする

セリフ例:

> 2週間で友だちが30人増えました。このペースだと3ヶ月で200人前後まで見えます

> 自動応答も使われていて、営業時間や予約導線の問い合わせがかなり整理されています

> ここから配信頻度と改善をこちらで回すなら、スタンダードの2.5万円が一番バランスいいです

> もしまずは小さく始めるなら、ライトの1.5万円からでも大丈夫です

ここで大事なのは、無理に詰めないこと。無料で使ってもらいながら成果が出れば、あとから月額化できるケースも多い。

## 6. よくある商談シーンと対応

| シーン | 対応 |
|--------|------|
| オーナーではなく店長が対応する場合 | 店長にデモを見せて「オーナーさんにもお見せしたい」と伝える。2回目アポでオーナー同席を狙う |
| 営業時間中で忙しそうな場合 | 5分に短縮する。デモだけ見せて「詳しくは改めて」で切り上げる |
| 「他のスタッフにも聞かないと」と言われた場合 | A4チラシを人数分渡す。デモアカウントのQRも共有する |
| 即決で「やりたい」と言われた場合 | その場で管理者権限の共有方法を案内し、翌日から構築を開始する |
| 「もう少し考えたい」と言われた場合 | 無理に押さない。1週間後にフォロー電話することだけ伝える |
| 既にLステップを使っている場合 | 無理に乗り換え提案しない。「今の環境を活かしつつ、配信内容の改善だけでも効果出ます」で入る |
| 月額を「高い」と言われた場合 | 食べログやホットペッパー等の月額と比較する。それでも厳しければライト1.5万を提案する |

この表のパターンはよく出る。事前に返しを決めておけば、商談中に詰まりにくい。

## 7. 商談管理テンプレート

商談ごとに最低限これだけ記録しておけば、あとから追いかけやすい。

```markdown
## [店名] — YYYY-MM-DD

- 業態:
- エリア:
- オーナー/店長名:
- パターン: A / B / C
- LINE友だち数:
- 現在の販促: （食べログ/ぐるなび/Instagram/なし）
- 月商感:
- 反応が良かったポイント:
- 懸念点:
- 結果: 構築合意 / 保留 / NG
- 次のアクション:
- 次回アポ日:
```

このテンプレは、商談後すぐ埋める。後でまとめて書こうとすると、反応の温度感が抜ける。
