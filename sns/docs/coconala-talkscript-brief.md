# ココナラ外注用：電話トークスクリプト作成依頼書

作成日: 2026-03-31

---

## 依頼概要

小規模店舗（飲食店・美容室・整体等）に対して、LINE公式アカウントの無料構築＋月額運用サポートを提案する**電話営業用トークスクリプト**の作成をお願いします。

---

## サービス概要

### 何を売るのか

「LINE公式アカウントの構築・運用代行サービス」です。

- **初期構築: 完全無料**（リッチメニュー、自動応答、ステップ配信、QR付きPOPまで全部込み）
- **月額運用サポート: 1.5万〜3万円**（配信代行、レポート、改善提案）
- **ツール利用料: 0円**（自社開発ツールのため追加のツール代が発生しない）

### 3つのプラン

| | ライト | スタンダード | プレミアム |
|---|---|---|---|
| 月額 | 1.5万円 | 2.5万円 | 3万円 |
| 配信代行 | 月2回 | 月4回 | 月8回 |
| リッチメニュー変更 | 月1回 | 月2回 | 月4回 |
| セグメント配信 | なし | あり | あり |
| 月次レポート | 数値のみ | 改善提案付き | 改善提案+次月計画 |
| 最低契約 | 3ヶ月 | 3ヶ月 | 3ヶ月 |

**狙い: スタンダード（2.5万円）に集める**

### 競合との違い

| | うち | Lステップ構築代行（相場） |
|---|---|---|
| 初期構築費 | 0円 | 10〜50万円 |
| ツール月額 | 0円 | 2〜3.3万円 |
| 運用代行月額 | 1.5〜3万円 | 5〜20万円 |
| 合計月額負担 | 1.5〜3万円 | 7〜23万円 |

## 電話営業の前提条件

### 電話のタイミング

- Googleマップで事前リサーチ済みの店舗に電話する
- 飲食店のアイドルタイム（14:00〜17:00）を狙う
- 美容室・整体は比較的終日対応可

### 電話の相手

- **個人経営〜数店舗の小規模店舗のオーナー or 店長**
- 年齢層: 30〜50代が多い
- ITリテラシー: 低め。「LINE公式アカウント」は聞いたことあるが使いこなせていない人が大半
- 忙しい。電話で長く話す余裕はない

### 電話相手の状態（3パターン）

| パターン | 割合 | 状態 | 刺さるポイント |
|---------|------|------|------------|
| A: アカウント未開設 | 3〜4割 | LINEで何ができるか知らない | 「初期費用ゼロで始められる」「ダメなら辞められる」 |
| B: 作ったけど放置 | 4〜5割 | アカウントはある、友だちも少しいる。でも放置 | 「今あるアカウントをそのまま活かせます」 |
| C: そこそこ使えている | 1割以下 | 月に数回配信はしている | 「今やっていることの効果を3倍にできます」 |

**一番多いのはパターンB。ここを主要ターゲットとしてスクリプトを設計してください。**

## トークスクリプトに求める構成

### 1. 受付突破（15秒）

- オーナー or 店長につないでもらうための冒頭トーク
- 「営業電話」と警戒されない切り出し方
- スタッフが出た場合と、オーナー直接の場合の2パターン

### 2. 導入・自己紹介（15〜20秒）

- 何者か、何の用件か、を簡潔に伝える
- 「無料」を早めに出す（ここで切られないようにする）
- 具体的な配信例を1〜2個挙げる（抽象的だと伝わらない）

### 3. ヒアリング（30秒〜1分）

- 相手の現状を聞き出す質問（LINE公式やってますか？ / 友だち何人くらいですか？ / 配信はされてますか？）
- パターンA/B/Cのどれかを素早く判定するための質問設計
- 相手が話しやすい質問の順番

### 4. 提案（1〜2分）

- パターンA/B/Cに応じた提案の分岐
- 具体的なメリットを業種別に伝える（飲食店なら雨の日クーポン、美容室なら次回予約リマインド等）
- 「初期費用ゼロ」「ツール代もゼロ」「合わなければそのまま無料で使い続けてOK」

### 5. クロージング（30秒）

- 電話のゴールは**対面アポ or オンライン説明の約束**（電話で契約を取るわけではない）
- 日時の提案方法（候補を2つ出す等）
- 「一度お店に伺って、実際の画面をお見せしながら5分だけ説明させてください」

### 6. 想定Q&A / 切り返しトーク

以下のよくある反応への返答を用意してください。

| 反応 | 補足 |
|------|------|
| 「忙しいので」 | 最も多い断り文句 |
| 「うちは小さい店だから必要ない」 | 小さい店ほどLINEが効く理由を伝えたい |
| 「LINEはもうやってるから大丈夫」 | パターンBの可能性大。現状を深掘りしたい |
| 「お金かかるんでしょ？」 | 初期無料→月額の流れを自然に伝えたい |
| 「月額いくら？」 | 正直に答えつつ「まず無料で試せる」に戻す |
| 「前にも似たような営業来た」 | Lステップ系との違いを簡潔に伝えたい |
| 「考えておきます」 | 次のアクションを約束して終わりたい |
| 「チラシだけ送って」 | メール or LINE で資料を送り、フォロー電話につなげたい |

## 業種別の具体例（スクリプトに組み込んでほしい）

電話で「たとえば…」と話すときに使う具体例です。

### 飲食店

- 雨の日に「今日だけ1杯無料」クーポンを送ると、空いてた席が埋まる
- 金曜の昼に「今週末のおすすめ」を配信すると週末の予約が増える
- 友だち100人に月2回送るだけで、月5〜10人の追加来店が見込める

### 美容室・サロン

- 来店後に「次回は○月頃がおすすめです」と送ると再来店率が上がる
- 指名スタイリストの紹介配信で指名率アップ
- 空き枠が出たときにLINEで告知すると即埋まる

### 整体・マッサージ

- 施術後にセルフケアのアドバイスを送ると信頼感が上がる
- 「前回から1ヶ月経ちました」のリマインドで定期来院につながる

### クリニック・歯科

- 定期健診のリマインド配信で来院率アップ
- 季節の健康情報（花粉、インフルエンザ等）で開封率が高い

## トーンとNGワード

### トーン

- 堅すぎず、フランクすぎず。**「近所の詳しい人」のような距離感**
- 上から教えるのではなく、「お手伝いします」のスタンス
- 飲食店オーナーは忙しいので、回りくどい説明はNG

### 使ってほしい言葉

- 「無料で作ります」「お金かかりません」
- 「一度作ったら勝手に動く仕組み」
- 「友だちに直接届く」
- 「常連さんにまた来てもらう仕組み」

### NGワード（使わないでほしい）

| NG | 理由 |
|----|------|
| CVR、LTV、セグメント等のマーケ用語 | オーナーに伝わらない |
| 「Lステップ」「エルメ」等のツール名 | 混乱するだけ。比較は聞かれたら答える程度 |
| 「導入していただく」等の堅い敬語 | 距離感が出る |
| 「売上が○倍に！」等の誇大表現 | 信頼を失う。具体的な数字で語る |
| 「今だけ」「期間限定」等の煽り | 安っぽくなる |

## 納品物に含めてほしいもの

1. **メインスクリプト**: 受付突破→導入→ヒアリング→提案→クロージングの一連の流れ
2. **分岐シナリオ**: パターンA/B/Cごとの提案トーク
3. **業種別アレンジ例**: 飲食店・美容室・整体の3パターン（メインスクリプトの「たとえば」部分を差し替えるだけで使える形）
4. **切り返しトーク集**: 上記8パターンの想定Q&Aへの返答
5. **電話後のフォローメッセージ文例**: 「先ほどお電話した○○です」的なLINE or メール文面（アポ取れた場合 / 資料送付の場合）
